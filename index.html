Base
Basic examples

FlexWork is simple tool with zero use of fancy technologies. It's common to apply this technics by hands. But this tool aims to simplify flexbox technics for people who want end-result and do not want to dig into details of CSS.

All examples a live so don't hesitate to resize window to see FlexWork in action.

Position of the individual blocks in a row

This allows the default alignment (or the one specified by flex-v-align-items--...) to be overridden for individual flex items. Available values:

  • flex-align--auto
  • flex-align--start
  • flex-align--end
  • flex-align--center
  • flex-align--baseline
  • flex-align--stretch








































                                
<div class="flex-box">
    <div class="flex-box flex-v-align-items--start">
        <div class="flex-align--end"></div>
        ...
        <div></div>
    </div>

    <div class="flex-box flex-v-align-items--end">
        <div class="flex-align--center"></div>
        ...
        <div></div>
    </div>

    <div class="flex-box flex-v-align-items--center">
        <div class="flex-align--baseline"></div>
        ...
        <div></div>
    </div>

    <div class="flex-box flex-v-align-items--baseline">
        <div class="flex-align--stretch"></div>
        ...
        <div></div>
    </div>

    <div class="flex-box flex-v-align-items--stretch">
        <div class="flex-align--start"></div>
        ...
        <div></div>
    </div>
</div>

Attention!

Note that float, clear and vertical-align have no effect on a flex item.
